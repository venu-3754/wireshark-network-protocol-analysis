# Executive Summary

## Project Overview

This project demonstrates practical network traffic analysis using Wireshark by capturing and analyzing multiple network protocols in a controlled lab environment.

The objective was to investigate how different protocols communicate across a network, identify insecure communication methods, and document findings using packet-level evidence.

## Objectives

- Capture live network traffic.
- Analyze HTTP communication.
- Demonstrate plaintext credential exposure over HTTP.
- Inspect DNS name resolution.
- Analyze TCP three-way handshake.
- Examine ARP request and reply communication.
- Document observations and security implications.

## Tools Used

- Wireshark 4.6
- Windows 11
- Python HTTP Server
- Npcap Loopback Adapter

## Protocols Covered

- HTTP
- DNS
- TCP
- ARP

## Outcome

The project successfully demonstrates protocol analysis, packet inspection, and identification of insecure network communication using Wireshark.
