# Recommendations

## HTTP

Use HTTPS instead of HTTP to encrypt web traffic.

---

## Credentials

Never transmit authentication credentials over plaintext HTTP.

---

## DNS

Use encrypted DNS technologies such as DNS over HTTPS (DoH) or DNS over TLS (DoT) where appropriate.

---

## Network Monitoring

Monitor abnormal ARP traffic to detect spoofing attacks.

---

## Secure Communication

Implement TLS encryption for all sensitive network services.

---

## Continuous Monitoring

Use packet analysis together with SIEM platforms such as Splunk for continuous monitoring and incident detection.
